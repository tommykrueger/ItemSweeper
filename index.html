How to play

Press the start button to begin the game.

Use the mouse button to reveal a square on the board.

There is a number of items that you have to find and a number of steps that you have available. If you run out of steps the game is over.

If you click on a square with an item in it will be collected to your item inventory

Try to find all items by analysing the numbers. If you manage to find all - you win.

The numbers show how many items are adjacent to that square. For example if you reveal a square and the number 3 appears it means there are 3 items neighboring that square.
